# Tank attack
#### Простая игра на Phaser 3

### Демо: https://phaser-game-tank.vercel.app/

Работает только на десктопе 🖥️. Управление: w,a,s,d + mouse.


## Stack
Typescript + Phaser.js + Vite + Howler


## Запуск

### Установить npm-зависимости
```
npm i
```
### Запустить сервер
```
npm run dev
```
### Открыть игру в браузере
http://localhost:5173/
